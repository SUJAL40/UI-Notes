	3) Bytes convert to chars   	< b > h  e l l o </ b >
	4) Chars convert to token    	<b> Token/Tag
		called tokenization 
	5) Token convert into elements  <b> Bold <a> Anchor
	6) Token are arranged in a 			parent
		hierarchical order called 	child1  child2
		DOM 
	7) DOM is converted into layout
	8) Layout undergo a process called Render
	9) Render will paint result on browser

	Meta SEO Friendly:

	<meta name="keywords" content="some keywords to find your page">
	<meta name="description" content="something about your page to show in SEO">
	
	Google bots read meta tags to understand webpage information.
	
		<meta charset="utf-8">

		UTF = Unicode Transformation Format
		8	= 8bit memory size required to read the chars.
			= 8bit is used for english

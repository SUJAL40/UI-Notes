FAQ: What are the issues with object?
- It can have keys only of "string" type.
- It uses all explicit methods and operators for manipulation.
- It is slow in interactions.

Note: Object provides a schema for data. [structured data]


								Map Type
								
- It is a key / value collection same as object.
- It provides schema less data means Key can be any type and value can be any type.
- It provides implicit methods for manipulation.
- It is faster when compared to object.

Syntax:

	var ref = new Map();

	.size		: It returns the length of keys
	.set()		: It adds a new key and value
	.get()		: It can access the value with reference of key
	.delete()	: It removes a keys
	.clear()	: It removes all keys
	.has()		: It finds a key
	.keys()		: It returns all keys
	.values()	: It returns all values
	.entries()	: It returns both keys and values


eg.

	var topics = new Map();
	topics.set(1, "HTML is a markup language");
	topics.set("CSS", "It configures styles");
	topics.set(true, "Stock Available");
	topics.set("Cities", ["Delhi", "Hyd"]);
	topics.set(1001, {Name:"TV", Rating:4.3});
	
	if(topics.has("CSS")){
		document.write(topics.get("CSS"));
	} else {
		document.write("Can't find cSS<br>");
	}

	for(var item of topics.entries()){
		document.write(item + "<br>");
	}
	
- Object is used for structured key-value data, while Map provides more flexible key-value storage with keys of any data type.

1. setTimeout()
- It is used to control "debounce".
- Bounce is a mechanism where the tasks are sent into process immediately one after
  another.
- Debounce is a mechanism where you can keep the tasks wating in memory and release
  after specific duration of time interval.

Syntax:
	setTimeout(function(){}, interval)

- we can write function outside and ony call in setTimeout()
- In normal clock 100ms = 1s
		CPU clock 1000ms = 1s

eg.

3. setlnterval()
- It loads the task into memory and releases a copy of task into process at regular time
 intervals.
- It performs the given task repeatedly, until removed from memory.

Syntax:
	setlnterval(function(){}, interval)

4. clearlnterval()
- It removes the task from memory, which is set to execute at regular intervals.

Syntax:

        function LoadProduct(id) {
            fetch(`https://fakestoreapi.com/products/${id}`)

                }).then(function (products) {

        function NextClick() {
            count++;
            LoadProduct(count);
        }
        function PrevClick() {
            count--;
            LoadProduct(count);
        }

        function LoadProductAuto() {
            count++;
            document.getElementById("rangePicker").value = count;
            fetch(`https://fakestoreapi.com/products/${count}`)
                .then(function (response) {
                    return response.json();
                }).then(function (products) {
                    document.getElementById("lblTitle").innerHTML = products.title;
                    document.getElementById("imgPoster").src = products.image;
                })

            thred = setInterval(LoadProductAuto, 1000);

            clearInterval(thred);
            document.getElementById("lblStatus").innerHTML = "Paused";

            LoadProduct(document.getElementById("rangePicker").value)
        }

    </script>
